# Palindromos

## Requisitos
* ASP.NET Core 5 o superior
* nodejs 14 o superior

## Instalación

### En el proyecto palindromos-nuxt
Ejecutar en terminal
```
npm install
```

## Ejecución
### En el proyecto palindromos-nuxt
Ejecutar en terminal
```
npm run dev
```

### En el proyecto PalindromosAspNetCore
Ejecutar en terminal
```
dotnet run
```
